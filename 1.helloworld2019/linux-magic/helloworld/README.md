# Simple Hello World program

This is a simple Hello World go program.

Requirements:
* [Go](https://golang.org/doc/install#install)
* [GNU make](https://www.gnu.org/gnu/gnu.html)

## Make targets

### 1.Run
```
make run
```

### 2.Compile
```
make simplemagic
```

### 3.Clean
```
make clean
```
